# Lensmor API Documentation

Documentation and examples for the official Lensmor API.

## Repository structure

- `concepts/` — conceptual guides and shared product knowledge
- `api-reference/` — endpoint reference documentation
- `docs.json` — Mintlify navigation and site configuration

## Local development

Preview the documentation site locally:

```bash
pnpm dlx mintlify dev
```

## Editing guidelines

- Update conceptual content in `concepts/`
- Update endpoint-specific content in `api-reference/`
- Update navigation and site metadata in `docs.json`

## Publishing

Publish the documentation through the repository's configured Mintlify deployment workflow.
